# Spark 4.1.2 完整接入设计

**日期**: 2026-06-25
**目标**: 恢复 pyspark.ml 原生聚类、拆分 spark_engine.py 模块、补充 SparkEngine 测试。

## 背景

Spark 4.1.2 Connect 完整支持 `pyspark.ml`（SPARK-50812），不再需要 sklearn 回退。当前 `spark_engine.py` 596 行，混杂了构造逻辑、数据清洗、分析方法和健康检查，需要按职责拆分以便测试。

## 设计

### 1. clustering 恢复 pyspark.ml

当前使用 sklearn（`toPandas` 拉全量 → `StandardScaler` → `KMeans` → `PCA`），改为全 Spark 实现：

```python
# 流程
stat = spark.read.parquet(HDFS_PROCESSED / "VideoStat")
features = ["view", "like", "coin", "favorite"]

assembler = VectorAssembler(inputCols=features, outputCol="features")
assembled = assembler.transform(stat)

scaler = StandardScaler(inputCol="features", outputCol="scaled_features",
                        withStd=True, withMean=True)
scaler_model = scaler.fit(assembled)
scaled = scaler_model.transform(assembled)

kmeans = KMeans(k=3, seed=42, featuresCol="scaled_features", predictionCol="label")
model = kmeans.fit(scaled)
predictions = model.transform(scaled)

# Silhouette
evaluator = ClusteringEvaluator(featuresCol="scaled_features", metricName="silhouette")
sil_score = evaluator.evaluate(predictions)

# PCA scatter data
pca = PCA(k=2, seed=42, inputCol="scaled_features", outputCol="pca_features")
pca_model = pca.fit(scaled)
pca_result = pca_model.transform(scaled)

# Cluster analysis — groupBy on predictions
labeled = predictions.select("aid", "label").join(stat, "aid", "inner")
cluster_agg = labeled.groupBy("label").agg(
    count("*").alias("cnt"),
    avg("view"), avg("like"), avg("coin"), avg("favorite"),
).collect()
```

### 2. 模块拆分

```
src/bilianalysis/engine/
├── spark_engine.py              # SparkEngine 类 (~100行)
│     __init__, _get_spark, _create_session,
│     _sync_raw_to_hdfs, _ensure_processed,
│     ping, ping_hdfs, model_comparison
│
├── spark/
│   ├── __init__.py
│   ├── clean.py                 # clean_data, _extract_tables, _fill_missing 等
│   ├── analysis.py              # statistics, clustering, prediction
│   └── _helpers.py              # HDFS_RAW/PROCESSED 常量, safe path getters
```

分析方法改为**纯函数**，接收 `(spark, paths)` 参数：

```python
# spark/analysis.py
def compute_statistics(spark: SparkSession, processed_path: str) -> StatReport: ...
def compute_clustering(spark: SparkSession, processed_path: str) -> ClusterReport: ...
def compute_prediction(spark: SparkSession, processed_path: str) -> PredictionReport: ...
```

`SparkEngine` 方法变成薄代理：

```python
def statistics(self) -> StatReport:
    return compute_statistics(self._get_spark(), self.HDFS_PROCESSED)
```
